## 📝 API Endpoints Summary

### Authentication
- `POST /api/auth/signup` - Register new user
- `POST /api/auth/login` - User login
- `GET /api/auth/me` - Get current user

### Devices
- `GET /api/devices` - List devices (with pagination & filters)
- `POST /api/devices` - Register device
- `GET /api/devices/{device_id}/status` - Device status
- `PATCH /api/devices/{device_id}` - Update device
- `DELETE /api/devices/{device_id}` - Delete device

### Alerts
- `GET /api/alerts` - List alerts (with filters)
- `POST /api/alerts` - Create alert
- `POST /api/alerts/{alert_id}/resolve` - Resolve alert

### Heartbeat
- `POST /api/heartbeat` - Device heartbeat

### Notification Preferences
- `GET /api/notification-preferences` - Get preferences
- `PUT /api/notification-preferences` - Update preferences

### Health
- `GET /api/health` - Health check
