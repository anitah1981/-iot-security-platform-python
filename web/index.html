What you can do now

Once you’ve created an account, you can add devices, receive alerts, and post device heartbeats.

  • Create a user: /api/auth/signup
  • Login: /api/auth/login
  • Create a device: /api/devices
  • Heartbeat: /api/heartbeat
  • View alerts: /api/alerts
