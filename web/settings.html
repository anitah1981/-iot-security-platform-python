📡 Device agent key

When to use this: Only if you run our optional discovery script (to find devices on your Wi‑Fi) or the device agent (to see devices as online/offline) on a computer on your network. Get the key here, then use it when you set up the script. If you only add and manage devices from this website or the phone app, you don't need this key.

Adding devices in this app always requires you to be logged in. The key is only for those optional scripts — they run on a separate computer (e.g. Raspberry Pi or PC) that has no browser and no access to your login, so we use the key to link their data to your account.
