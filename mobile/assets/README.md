# Mobile App Assets

This folder contains app icons and images.

## Required Assets

The following assets are referenced in `app.json`:

- `icon.png` - App icon (1024x1024px)
- `splash.png` - Splash screen (1242x2436px recommended)
- `adaptive-icon.png` - Android adaptive icon foreground (1024x1024px)
- `favicon.png` - Web favicon (48x48px)
- `notification-icon.png` - Notification icon (96x96px)

## Quick Solution

For now, Expo will use default icons if these files don't exist. You can:

1. **Use Expo's default icons** (works for testing)
2. **Create your own** using a tool like:
   - https://www.appicon.co/
   - https://www.canva.com/
   - https://www.figma.com/

## For Production

Create proper branded icons before submitting to app stores.
