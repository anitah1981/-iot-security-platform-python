# 🚀 Build Mobile Apps Now

**Everything is configured - just run these commands:**

---

## ⚡ **Quick Start**

### **1. Install EAS CLI**
```bash
npm install -g eas-cli
```

### **2. Login**
```bash
eas login
```
(Create free account at https://expo.dev if needed)

### **3. Configure**
```bash
eas build:configure
```
(Press Enter for defaults)

### **4. Update Backend URL**

**Edit `app.json` - change this line:**
```json
"apiUrl": "http://localhost:8000"
```

**To your actual backend URL:**
```json
"apiUrl": "https://your-backend-url.com"
```

### **5. Build iOS**
```bash
eas build --platform ios --profile preview
```

### **6. Build Android**
```bash
eas build --platform android --profile preview
```

---

## ✅ **What Happens**

1. **EAS uploads** your code to cloud
2. **Builds** iOS and Android apps (10-15 min each)
3. **Gives you** download links
4. **Install** on your devices

---

## 📱 **Installation**

**iOS:**
- Get link from EAS output
- Open on iPhone
- Install (may need to trust developer)

**Android:**
- Download APK file
- Install on device
