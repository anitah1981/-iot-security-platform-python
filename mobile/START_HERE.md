## ⚡ **3 Simple Steps**

### **1. Login to Expo** (2 min)
```bash
cd mobile
eas login
```
*(Create free account at https://expo.dev if needed)*

### **2. Update API URL** (1 min)
Edit `app.json` - change `"apiUrl": "http://localhost:8000"` to your backend URL

**For local testing:** Use ngrok (see RUN_BUILD_NOW.md)  
**For production:** Use your deployed backend URL

### **3. Build Apps** (10-15 min)
```bash
.\build-apps.ps1
```

**Done!** You'll get download links for iOS and Android apps.
