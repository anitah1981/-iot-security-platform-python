# 🚀 Quick Start - Windows

**Get your mobile app running on Windows in 5 minutes!**

---

## ⚡ **Fastest: Physical Android Device**

### **1. Install Expo Go on Your Phone**
- Download from Google Play Store
- Search: "Expo Go"

### **2. Connect to Same WiFi**
- Make sure your phone and computer are on the same WiFi network

### **3. Start Expo**
```bash
cd mobile
npm start
```

### **4. Scan QR Code**
- Open Expo Go app
- Tap "Scan QR code"
- Point camera at QR code in terminal
- App loads on your phone!

**Done!** ✅

---

## 🖥️ **Alternative: Android Emulator**

### **1. Install Android Studio**
- Download: https://developer.android.com/studio
- Install with default settings

### **2. Create Emulator**
- Open Android Studio
- **Tools** → **Device Manager**
- **Create Device** → Choose Pixel 5
- Download Android 13 system image
- **Finish**

### **3. Start Emulator**
- Click **Play** button next to your device
- Wait for emulator to boot

### **4. Start Expo**
```bash
cd mobile
npm start
# Press 'a' to open Android
```

**Done!** ✅

---

## 🌐 **Web Version (Quick Test)**

```bash
cd mobile
npm start
# Press 'w' to open web
```

Opens in browser - good for quick UI testing!

---

## ❌ **iOS Simulator Not Available**

**Xcode is macOS only** - you can't run iOS Simulator on Windows.

**Options:**
- ✅ Use Android Emulator (recommended)
- ✅ Use physical Android device
- ✅ Use physical iOS device (if you have one)
- ✅ Use web version

---

## 🎯 **Recommended Path**

1. **Start with physical Android device** (fastest, 2 minutes)
2. **Set up Android emulator** (better for development, 15 minutes)
3. **Test on physical iOS device** (if you have one)

---

**You're ready to test!** 📱
