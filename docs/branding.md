# Brand / UI Theme (Security + Trust)

This project uses a “calm security” theme:
- Deep navy backgrounds (serious, stable)
- Trusted blue primary actions (confidence)
- Teal as verification/accent (clarity, “secure” cues)
- High contrast for accessibility

## Color tokens
Defined in `web/assets/app.css`:
- `--bg`: deep navy
- `--surface`, `--card`: elevated panels
- `--primary`, `--primary-2`: trusted blues
- `--accent`: teal verification
- `--ok`, `--warning`, `--danger`: status colors

## Typography
- System UI stack (fast, familiar, trustworthy)
- Clear hierarchy: short headings + muted supporting text

## UI Principles (for consumers)
- Always show “what’s happening now” (online/offline, last seen)
- Make critical actions obvious (primary button) but not alarming
- Use consistent status badges and colors across web + mobile
