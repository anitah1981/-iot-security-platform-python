## ЁЯФз **Configuration**

### **1. API URL**
Edit `mobile/app.json`:
```json
{
  "expo": {
    "extra": {
      "apiUrl": "https://your-api-domain.com"
    }
  }
}
```

### **2. App Icons** (Optional)
Replace files in `mobile/assets/`:
- `icon.png` (1024x1024)
- `splash.png` (1242x2436)
- `adaptive-icon.png` (1024x1024)
